# 🌐 Network Project 1 - Socket Programming

## 📌 Overview
This project was developed for the **ENCS3320 - Computer Networks** course at Birzeit University.  
It focuses on applying networking concepts through practical implementation using **socket programming**.

The project includes:
- Network analysis using command-line tools and Wireshark
- A TCP-based web server
- A UDP-based multiplayer trivia game

According to the project report :contentReference[oaicite:0]{index=0}, the system demonstrates real-world client-server communication and HTTP request handling.

---

## ⚙️ Technologies Used
- Python 🐍
- Socket Programming (TCP & UDP)
- HTML & CSS
- Wireshark
- Command Line Tools (ping, tracert, nslookup, telnet)

---
## 📊 Key Learning Outcomes
Understanding TCP vs UDP
Building real client-server systems
Handling HTTP requests manually
Working with sockets and networking protocols
Improving teamwork and problem-solving skills
